Platform Overview

{{ config.APP_DISPLAY_NAME }}

This application captures login attempts, enriches them with public-IP-based location data when available, scores risk, detects brute-force behavior, and exposes both analyst UI and JSON APIs.

How detection works

Repeated failures from the same public IP or against the same username inside the configured window generate a critical alert and temporary IP block.

Data model

Each login attempt stores public IP, optional private IP, IP version, source, country, country code, region, city, timezone, ISP, and optional coordinates.

Deployment model

The app supports local file watching for traditional deployments and request-driven ingestion for serverless deployments such as Vercel.
